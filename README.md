# Road-Network-Detection-Using-OpenCV
# Description

This project demonstrates a beginner-level Computer Vision pipeline for detecting and highlighting road-like structures from satellite imagery using OpenCV and Python.
# Objective

Detecting road structures from satellite imagery using classical image processing.

# Technologies Used
Python
OpenCV
NumPy
Matplotlib


# Processing Pipeline

Satellite Image
→ RGB Conversion
→ Grayscale Conversion
→ Gaussian Blur
→ Canny Edge Detection
→ Dilation
→ Contour Detection
→ Bounding Box Localization
